Related Repositories

Vaibify is designed to work with the VPLanet ecosystem of tools for simulating planetary evolution. Each repository solves a specific piece of the analysis pipeline.

Repository

Description

VPLanet

The Virtual Planet Simulator – a C code that models planetary system evolution by coupling physics modules.

vplot

Plotting package for VPLanet output with standardized figures and accessible colors.

vspace

Generate parameter sweeps of VPLanet input files.

multiplanet

Run VPLanet parameter sweeps in parallel across multiple cores.

bigplanet

Compress large VPLanet output suites into indexed HDF5 archives.

alabi

Machine learning surrogate model for fast Bayesian posterior inference.

vconverge

Derive converged probability distributions for VPLanet output parameters.

MaxLEV

Maximum likelihood estimator for VPLanet simulations.

vplanet_inference

Interface between VPLanet and Bayesian inference packages (alabi, emcee, dynesty).

The planetary template for vaibify init pre-configures all of these repositories in a single container. See Templates for details.
